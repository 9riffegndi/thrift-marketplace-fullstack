# Thrift Marketplace Fullstack

Modern C2C Marketplace and Swap Platform for Premium Thrifting.

## Architecture
- **Frontend**: Next.js (React) - In `recloth-fe/`
- **Backend**: Laravel (PHP) - In `recloth-be/`

## Features
- Premium Minimalist UI (H&M Inspired)
- Real-time Chat & Notifications
- Product Swapping System
- Wallet & Integrated Payments

## Getting Started
1. Run `setup.bat` to install all dependencies.
2. Run `run-all.bat` to start dev servers.
