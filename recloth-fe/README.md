# Recloth - Frontend

Premium Minimalist Storefront for Recloth Marketplace. Inspired by high-end retail aesthetics (H&M style).

## Tech Stack
- **Framework**: Next.js 15+
- **Styling**: Tailwind CSS 4+
- **Icons**: Lucide React
- **State Management**: Zustand
- **Real-time**: Laravel Echo & Socket.io-client
- **Data Fetching**: React Query

## Key Features
- **Modern UI**: Clean, high-contrast, pure white & black aesthetic.
- **Dynamic Catalog**: Category filtering and real-time search.
- **Enhanced Chat**: WhatsApp-style interface with delivery status.
- **Swap Portal**: Unique interface for managing product exchanges.
- **Premium Checkout**: Streamlined 1-click buy and wallet integration.

## Installation
```bash
npm install
```

## Running
```bash
npm run dev
```

## Structure
- `/src/app`: Next.js App Router pages.
- `/src/components`: Reusable UI components.
- `/src/hooks`: Custom React hooks for API interaction.
- `/src/lib/store`: Zustand stores.
